# Chapter 12: Limits and Colimits — 극한과 쌍대극한

> **핵심 개념**: **극한(Limit)**은 곱(product)의 일반화다. 임의의 다이어그램(펑터)에 대해 **뿔(cone)**을 정의하고, 보편적인 뿔이 극한이다. 자연 변환을 사용하면 극한을 깔끔하게 정의할 수 있다. 화살표를 뒤집으면 **쌍대극한(Colimit)**을 얻는다.

---

## 곱에서 극한으로

곱(product)의 보편적 구성을 일반화하자. 곱은 두 대상 a, b를 선택하고, 투영을 가진 대상 중 최적을 찾는 것이었다.

"두 대상을 선택하는 것"을 범주적으로 재해석하면: **카테고리 2**(대상 2개, 사상 없음)에서 카테고리 C로의 **펑터 D**이다. 이 펑터가 a, b를 "선택"한다.

이 관점을 확장하면, 임의의 인덱스 카테고리 **I**에서 C로의 펑터 D가 **다이어그램(diagram)**이 된다.

---

## 뿔 (Cone)

다이어그램 D: I → C에 대한 **뿔(cone)**은:

- **꼭짓점(apex)**: 카테고리 C의 대상 c
- **각 다리(legs)**: c에서 다이어그램의 각 대상 D(i)로의 사상들 `π_i :: c → D(i)`
- 다이어그램의 모든 사상과 **가환(commute)**해야 함

```
-- I에 사상 f :: i → j가 있으면
-- D(f) ∘ π_i = π_j 를 만족해야 한다
```

곱은 가장 단순한 경우다: 인덱스 카테고리에 대상만 2개 있고 사상이 없는 경우.

### 뿔을 자연 변환으로

꼭짓점 c를 상수 펑터 Δ_c로 생각하면 (I의 모든 대상을 c로, 모든 사상을 id_c로 보내는 펑터), 뿔의 각 다리는 **자연 변환 α: Δ_c → D**의 성분이 된다.

자연성 조건: I의 사상 `f :: i → j`에 대해

```
D(f) ∘ α_i = α_j ∘ Δ_c(f)
D(f) ∘ α_i = α_j ∘ id_c
D(f) ∘ α_i = α_j
```

이것이 바로 뿔의 가환 조건이다!

---

## 극한의 정의

**극한(Limit)**은 **보편적인 뿔**이다:

> 다이어그램 D에 대한 극한은 뿔 (c, α)로, 다른 임의의 뿔 (c', α')에 대해 **유일한 사상** `m :: c' → c`가 존재하여 모든 i에 대해 `α'_i = α_i ∘ m`

```
      m
c' ------→ c
 \         / \
  α'_i   α_i  α_j
   \     /     \
    ↓   ↓      ↓
    D(i) --→ D(j)
         D(f)
```

### 극한의 예시들

| 다이어그램 | 극한 |
|-----------|------|
| 대상 2개, 사상 없음 | 곱 (Product) |
| 대상 2개, 사상 2개 (같은 방향) | 이퀄라이저 (Equalizer) |
| 빈 다이어그램 | 끝 대상 (Terminal Object) |
| 임의의 다이어그램 | 일반적 극한 |

### 극한을 자연 동형으로

극한은 자연 동형(natural isomorphism)으로도 표현된다:

```
C(c, Lim D) ≅ Nat(Δ_c, D)
```

"c에서 극한으로의 사상"은 "상수 펑터 Δ_c에서 D로의 자연 변환"과 1:1 대응된다.

---

## 쌍대극한 (Colimit)

극한의 쌍대. 화살표를 뒤집으면 된다:

- 뿔(cone) → **쌍대뿔(cocone)**: 다이어그램에서 꼭짓점으로의 사상
- 극한(limit) → **쌍대극한(colimit)**: 보편적인 쌍대뿔

| 다이어그램 | 쌍대극한 |
|-----------|---------|
| 대상 2개, 사상 없음 | 쌍대곱 (Coproduct) |
| 대상 2개, 사상 2개 (같은 방향) | 코이퀄라이저 (Coequalizer) |
| 빈 다이어그램 | 시작 대상 (Initial Object) |

---

## Haskell에서의 극한과 쌍대극한

```haskell
-- 곱 = 가장 간단한 극한
type Product a b = (a, b)

-- 쌍대곱 = 가장 간단한 쌍대극한
type Coproduct a b = Either a b

-- 이퀄라이저: f와 g가 같은 값을 내는 입력의 부분집합
-- equalizer f g = { x | f x == g x }
```

---

## 정리

- **다이어그램**: 인덱스 카테고리 I에서 C로의 펑터 D
- **뿔(Cone)**: 꼭짓점 c에서 다이어그램의 각 대상으로의 가환 사상들. 자연 변환 Δ_c → D
- **극한(Limit)**: 보편적인 뿔. 다른 모든 뿔에서 유일한 사상이 존재
- **극한의 예**: 곱(product), 이퀄라이저(equalizer), 끝 대상(terminal)
- **쌍대극한(Colimit)**: 극한의 쌍대. 쌍대곱, 코이퀄라이저, 시작 대상
- `C(c, Lim D) ≅ Nat(Δ_c, D)` — 극한의 자연 동형 표현
- 곱과 쌍대곱은 극한과 쌍대극한의 가장 단순한 경우
