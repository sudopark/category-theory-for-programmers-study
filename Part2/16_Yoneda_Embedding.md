# Chapter 16: Yoneda Embedding — 요네다 매장

> **핵심 개념**: **요네다 매장(Yoneda Embedding)**은 카테고리 C를 펑터 카테고리 [C^op, Set]에 **충실하고 가득한(fully faithful)** 방식으로 매장한다. 이를 통해 모든 카테고리를 Set 값 펑터의 카테고리 안에서 연구할 수 있다. 요네다 보조정리의 직접적인 귀결이며, "대상은 사상으로 완전히 결정된다"는 철학의 정점이다.

---

## 홈 펑터에서 요네다 매장으로

요네다 보조정리에서 F를 다른 홈 펑터 `C(b, -)`로 놓으면:

```
Nat(C(a, -), C(b, -)) ≅ C(b, a)
```

**"두 홈 펑터 사이의 자연 변환의 집합은 두 표현 대상 사이의 사상 집합과 동형이다."**

이것은 대상 a를 펑터 `C(a, -)`에 대응시키는 매핑이 사상까지 보존한다는 뜻이다. 이 매핑이 **요네다 매장**이다.

---

## 요네다 매장의 정의

**요네다 매장** Y: C^op → [C, Set]은 다음과 같이 정의된다:

- **대상 매핑**: C의 대상 a를 홈 펑터 `C(a, -)` (a에서 나가는 사상들의 펑터)에 대응
- **사상 매핑**: C의 사상 `f :: a → b`를 자연 변환 `C(f, -) :: C(b, -) → C(a, -)`에 대응

> 주의: a에서 b로의 사상이 C(b,-)에서 C(a,-)로의 자연 변환에 대응 — **방향이 뒤집힌다**. 이것이 C^op인 이유다.

반변 버전(공변 요네다 매장): a ↦ `C(-, a)`

---

## 충실하고 가득한 펑터 (Fully Faithful Functor)

펑터 F: C → D가 **충실(faithful)**하다는 것은 홈셋 사이의 사상 매핑이 단사(injective)라는 뜻이다:

```
F_{a,b} :: C(a,b) → D(Fa, Fb) 가 단사
```

**가득찬(full)** 펑터는 이 매핑이 전사(surjective)라는 뜻이다.

**충실하고 가득한(fully faithful)** 펑터는 홈셋 사이의 동형을 유도한다:

```
C(a, b) ≅ D(Fa, Fb)
```

요네다 매장은 **충실하고 가득하다**:

```
C(a, b) ≅ Nat(C(a, -), C(b, -))
```

이것이 의미하는 바: 대상 a, b 사이의 사상은 홈 펑터 C(a,-), C(b,-) 사이의 자연 변환과 **정확히 같다**. 정보 손실이 없다.

---

## Haskell에서의 요네다 매장

```haskell
-- 요네다 매장: a ↦ (a ->)  (Reader a 펑터)
-- 사상 매핑: f :: a -> b ↦ 자연 변환 (b -> x) -> (a -> x)
--          즉, precomposition: h ↦ h . f

-- 요네다 보조정리의 특수한 경우:
-- Nat((a ->), (b ->)) ≅ (b -> a)
-- 즉, Reader a에서 Reader b로의 자연 변환은 (b -> a)와 동형
```

이것은 역방향이다: `a → b`가 아니라 `b → a`. 공변 요네다 매장에서는:

```haskell
-- 공변 버전: a ↦ (-> a)  (Op a 반변 펑터)
-- Nat((-> a), (-> b)) ≅ (a -> b)
```

---

## 요네다 매장의 의미

### 1. 대상 = 홈 펑터

요네다 매장이 충실하고 가득하므로, 카테고리 C는 펑터 카테고리 [C^op, Set]의 **부분 카테고리**와 동치다. 모든 대상은 홈 펑터로 완전히 대체할 수 있다.

### 2. 대상의 외재적 정의

대상의 "정체성"은 내부 구조가 아니라 **다른 모든 대상과의 관계(사상)**로 결정된다. 두 대상이 모든 대상에 대해 같은 사상 패턴을 가지면, 두 대상은 동형이다.

### 3. Presheaf 카테고리

[C^op, Set]을 **presheaf 카테고리**라 한다. 요네다 매장은 C를 presheaf 카테고리에 매장한다. Presheaf 카테고리는 매우 좋은 성질을 가지므로 (모든 극한과 쌍대극한이 존재), 원래 카테고리를 연구하기 좋은 환경을 제공한다.

---

## 요네다 보조정리 + 매장의 조합

| 결과 | 의미 |
|------|------|
| `Nat(C(a,-), F) ≅ F(a)` | 요네다 보조정리: 자연 변환은 F(a)의 원소로 결정 |
| `Nat(C(a,-), C(b,-)) ≅ C(b,a)` | 보조정리의 특수 경우: 홈 펑터 간 자연 변환 = 사상 |
| Y: C^op → [C, Set] 충실가득 | 요네다 매장: 정보 손실 없이 매장 |
| `C(a,-) ≅ C(b,-)` ⟹ `a ≅ b` | 대상은 사상으로 완전히 결정됨 |

---

## 정리

- **요네다 매장**: 대상 a를 홈 펑터 C(a, -)에 대응시키는 충실하고 가득한 펑터
- `C(a, b) ≅ Nat(C(a,-), C(b,-))` — 사상과 자연 변환이 1:1 대응
- **충실하고 가득한(fully faithful)**: 홈셋 사이의 동형 유도. 정보 손실 없음
- **핵심 귀결**: `C(a,-) ≅ C(b,-)` ⟹ `a ≅ b` — 대상은 외부 관계로 결정
- **Presheaf 카테고리** [C^op, Set]: 카테고리를 매장하기 좋은 완전한 환경
- 요네다의 철학: 대상 자체를 알 필요 없다. 대상이 다른 대상과 **어떻게 관계하는가**가 전부다
