# Chapter 18: Adjunctions — 수반

> **핵심 개념**: **수반(Adjunction)**은 두 펑터 L: C → D와 R: D → C 사이의 관계로, `D(L a, b) ≅ C(a, R b)` (홈셋의 자연 동형)으로 정의된다. L을 **좌수반(left adjoint)**, R을 **우수반(right adjoint)**이라 하며, `L ⊣ R`로 쓴다. 수반은 동형보다 약하지만 동등보다 강한 "대략적 역관계"이며, 카테고리 이론의 가장 핵심적인 개념 중 하나다.

---

## 동등, 동형, 수반

"같다"의 강도를 비교하면:

| 개념 | 강도 | 의미 |
|------|------|------|
| 동등(Equality) | 가장 강함 | 구별 불가능 |
| 동형(Isomorphism) | 강함 | 뒤집을 수 있는 사상 존재 |
| 동치(Equivalence) | 중간 | 펑터의 합성이 동형에 가까움 |
| **수반(Adjunction)** | 약함 | 홈셋 수준에서 대응 |

동형은 `g ∘ f = id`와 `f ∘ g = id`를 요구하지만, 수반은 더 약한 조건을 만족한다.

---

## 수반의 정의: 홈셋 동형

두 카테고리 C, D 사이의 펑터 L: C → D와 R: D → C에 대해, **수반 L ⊣ R**은:

```
D(L a, b) ≅ C(a, R b)
```

이 동형이 a와 b 모두에 대해 **자연적(natural)**이어야 한다.

직관: "L로 변환한 후 D에서의 사상"과 "C에서의 사상 후 R로 변환한 것"이 1:1 대응된다.

---

## 단위와 쌍대단위 (Unit and Counit)

수반에서 특별한 자연 변환 두 개가 나온다:

### 단위 (Unit): η :: Id_C → R ∘ L

홈셋 동형에서 b = L a로 놓으면:

```
D(L a, L a) ≅ C(a, R(L a))
```

왼쪽에서 `id_{La}`를 골라내면 오른쪽에서 `η_a :: a → R(L a)`를 얻는다.

### 쌍대단위 (Counit): ε :: L ∘ R → Id_D

홈셋 동형에서 a = R b로 놓으면:

```
D(L(R b), b) ≅ C(R b, R b)
```

오른쪽에서 `id_{Rb}`를 골라내면 왼쪽에서 `ε_b :: L(R b) → b`를 얻는다.

```haskell
-- Haskell에서
eta   :: a -> R (L a)       -- 단위
epsilon :: L (R b) -> b     -- 쌍대단위
```

---

## 삼각 항등식 (Triangle Identities)

단위 η와 쌍대단위 ε는 **삼각 항등식**을 만족한다:

```
ε_{La} ∘ L(η_a) = id_{La}    -- L에 대한 삼각 항등식
R(ε_b) ∘ η_{Rb} = id_{Rb}    -- R에 대한 삼각 항등식
```

직관: L로 갔다가 η로 R∘L로 돌아온 후 ε로 내려오면 원래의 L 값으로 돌아온다. 완전한 역은 아니지만 "왕복 후 원점 복귀"가 가능하다.

---

## 수반의 예시들

### 커링 수반: (- × a) ⊣ (a →)

```
C(b × a, c) ≅ C(b, a → c)     -- 커링!
```

- L = (- × a): 곱 펑터
- R = (a →): 지수 펑터
- 이 수반이 바로 **커링**의 범주적 표현

### 자유-망각 수반: F ⊣ U

```
Mon(F(S), M) ≅ Set(S, U(M))
```

- L = F: 자유 펑터 (집합 → 자유 모노이드)
- R = U: 망각 펑터 (모노이드 → 기저 집합)

### 곱-대각선 수반

```
C×C(Δ a, (b, c)) ≅ C(a, b × c)
```

곱의 보편적 성질이 수반으로 표현된다.

---

## 수반에서 모나드가 나온다

수반 L ⊣ R에서 합성 **T = R ∘ L**은 자기 함자(endofunctor)이며, η와 μ = R ε L을 갖추면 **모나드**가 된다:

```haskell
-- T = R ∘ L
-- η :: a → T a                    (단위)
-- μ :: T (T a) → T a              (곱, R(ε_{La})로 유도)
```

모든 수반은 모나드를 만들고, 모든 모나드는 어떤 수반에서 유래한다 (역도 성립).

---

## 정리

- **수반** L ⊣ R: `D(L a, b) ≅ C(a, R b)` — 홈셋의 자연 동형
- **단위** η: Id → R∘L, **쌍대단위** ε: L∘R → Id
- **삼각 항등식**: η와 ε가 만족하는 일관성 조건
- **커링**: 곱 펑터와 지수 펑터 사이의 수반
- **자유-망각**: 자유 구성은 망각 펑터의 좌수반
- **수반 → 모나드**: T = R∘L은 모나드, η는 return, μ는 join에 해당
- 수반은 "대략적인 역관계"로, 카테고리 이론의 핵심 도구
