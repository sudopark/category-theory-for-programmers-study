# Chapter 25: Algebras for Monads — 모나드의 대수

> **핵심 개념**: 모나드 T에 대한 **T-대수(T-Algebra)**는 F-대수의 특수한 경우로, 일관성 조건(모나드 법칙과의 호환)을 추가로 만족해야 한다. T-대수의 카테고리는 **아일렌베르크-무어 카테고리(Eilenberg-Moore category)**라 하며, 이것과 **클라이슬리 카테고리**가 같은 모나드를 생성하는 수반의 양 극단이다.

---

## T-대수의 정의

모나드 (T, η, μ)에 대한 **T-대수**는 다음의 쌍이다:

- **운반 대상** a
- **구조 사상** `alg :: T a → a`

그리고 두 가지 **일관성 조건**을 만족해야 한다:

### 1. 단위 조건

```
alg ∘ η_a = id_a
```

다이어그램:
```
     η_a
a ------→ T a
 \         |
  \        alg
   id      |
    \      ↓
     ---→ a
```

### 2. 결합 조건

```
alg ∘ μ_a = alg ∘ T(alg)
```

다이어그램:
```
T(T a) --T(alg)-→ T a
  |                 |
  μ_a              alg
  |                 |
  ↓                 ↓
 T a ----alg----→  a
```

직관: η로 감싼 것을 평가하면 원래 값, 이중으로 감싼 것을 풀 때 순서 무관.

---

## Haskell에서의 T-대수

```haskell
-- 모나드 m에 대한 T-대수
type MAlgebra m a = m a -> a

-- 조건:
-- alg . return = id              (단위)
-- alg . join = alg . fmap alg    (결합)
```

### 예시: List 모나드의 대수

List 모나드에서 T-대수 `[a] → a`는 어떤 것이 있을까?

```haskell
-- 숫자 리스트를 합으로 평가
sumAlg :: [Int] -> Int
sumAlg = sum

-- 단위 조건: sum [x] = x ✓ (return x = [x])
-- 결합 조건: sum (concat xss) = sum (map sum xss) ✓
```

이것은 사실상 **모노이드**다! List 모나드의 T-대수는 모노이드와 동등하다.

### 예시: Maybe 모나드의 대수

```haskell
-- Maybe 대수: Maybe a → a는 기본값이 필요
maybeAlg :: a -> Maybe a -> a
maybeAlg def Nothing  = def    -- 기본값 제공
maybeAlg _   (Just x) = x     -- 값이 있으면 그대로

-- 단위 조건: maybeAlg def (Just x) = x ✓
```

## 아일렌베르크-무어 카테고리

모나드 T에 대한 모든 T-대수와 T-대수 준동형이 이루는 카테고리를 **아일렌베르크-무어 카테고리(Eilenberg-Moore category)** **C^T**라 한다.

- **대상**: T-대수 (a, alg :: T a → a)
- **사상**: T-대수 준동형 — `alg' ∘ T f = f ∘ alg`를 만족하는 f :: a → b

### 자유 T-대수

T-대수 중에서 "자유로운" 것이 있다:

```haskell
-- 자유 T-대수: 운반 대상이 T a이고 구조 사상이 μ_a인 것
-- μ :: T(T a) → T a
-- 단위 조건: μ . η_{Ta} = id_{Ta} ✓ (모나드 법칙)
-- 결합 조건: μ . μ_{Ta} = μ . T(μ_a) ✓ (모나드 법칙)
```

## 모나드와 수반의 관계

모든 모나드는 수반에서 유래한다. 주어진 모나드에서 수반을 복원하는 방법이 두 가지 있다:

### 1. 클라이슬리 카테고리 (최소)

- 대상: C의 대상과 동일
- 사상: 클라이슬리 화살표 `a → T b`
- 이것이 가장 "작은" 수반

### 2. 아일렌베르크-무어 카테고리 (최대)

- 대상: T-대수들
- 사상: T-대수 준동형
- 이것이 가장 "큰" 수반

두 수반 모두 동일한 모나드 T를 생성한다. 주어진 모나드를 생성하는 모든 수반은 이 두 극단 사이에 있다.

```
클라이슬리 카테고리 ←── 모든 수반 ──→ EM 카테고리
   (최소)                                (최대)
```

---

## 비교와 연결

| F-대수 | T-대수 |
|--------|--------|
| 임의의 펑터 F | 모나드 T |
| `F a → a` | `T a → a` |
| 추가 조건 없음 | 단위/결합 일관성 필요 |
| 시작 대수 → 카타모피즘 | 자유 대수 → 모나드의 bind |

T-대수는 F-대수에 모나드의 법칙과 호환되는 조건을 추가한 것이다.

---

## 정리

- **T-대수**: 모나드 T에 대한 대수. `T a → a` + 단위/결합 일관성 조건
- **단위 조건**: `alg ∘ η = id` — return으로 감싼 것을 평가하면 원래 값
- **결합 조건**: `alg ∘ μ = alg ∘ T(alg)` — 이중 감싸기를 풀 때 순서 무관
- **아일렌베르크-무어 카테고리** C^T: T-대수의 카테고리
- **List 모나드의 T-대수 = 모노이드**: 리스트를 접는 방법 = 이항 연산 + 단위
- **모나드 → 수반**: 클라이슬리(최소)와 EM(최대) 사이에 모든 수반이 존재
- 자유 T-대수의 구조 사상은 μ(join) 그 자체
