# Chapter 29: Topoi — 토포스

> **핵심 개념**: **토포스(Topos)**는 집합론의 대안적 기반을 제공하는 카테고리다. 토포스는 **부분 대상 분류자(subobject classifier)** Ω를 가지며, 집합의 카테고리 Set은 토포스의 대표적 예시다. 토포스에서의 논리는 고전 논리가 아닌 **직관주의 논리(intuitionistic logic)**이며, 이는 프로그래밍의 타입 시스템과 깊이 연결된다.

---

## 토포스란?

**토포스(topos, 복수형 topoi)**는 다음을 만족하는 카테고리다:

1. **모든 유한 극한**이 존재 (끝 대상, 곱, 이퀄라이저 등)
2. **지수 대상**이 존재 (데카르트 닫힌 카테고리)
3. **부분 대상 분류자** Ω가 존재

Set은 토포스다. 하지만 Set 외에도 무수히 많은 토포스가 있다.

---

## 부분 대상 분류자 (Subobject Classifier)

Set에서 집합 a의 **부분 집합**은 특성 함수(characteristic function)로 표현 가능하다:

```haskell
χ_S :: a -> Bool
χ_S x = if x ∈ S then True else False
```

일반 카테고리에서 이를 추상화하면:

> **부분 대상 분류자** Ω는 끝 대상 1에서 Ω로의 사상 `true :: 1 → Ω`와 함께, 모든 단사 사상(부분 대상) `m :: S → A`에 대해 **유일한** 사상 `χ :: A → Ω`가 존재하여 다음 풀백(pullback) 다이어그램이 가환:

```
S ----→ 1
|        |
m       true
|        |
↓        ↓
A ---χ-→ Ω
```

### Set에서

- Ω = {True, False} = Bool
- true :: () → Bool, true = const True
- χ는 부분 집합의 특성 함수

### 다른 토포스에서

Ω가 반드시 {True, False}일 필요는 없다! 예를 들어:
- **Presheaf 토포스**: Ω는 "체(sieve)"의 집합으로, 다양한 "참의 정도"를 가질 수 있다
- 이것이 직관주의 논리와 연결되는 이유

---

## 토포스와 논리

토포스의 내부 논리는 **직관주의 논리(intuitionistic logic)**다:

| 고전 논리 | 직관주의 논리 |
|-----------|-------------|
| 배중률: p ∨ ¬p 항상 참 | 배중률이 성립하지 않을 수 있음 |
| ¬¬p = p | 이중 부정 제거 안 됨 |
| Bool = {True, False} | Ω가 두 값 이상일 수 있음 |

프로그래밍과의 연결 (Curry-Howard):
- **고전 논리** ↔ continuation을 포함한 타입 시스템
- **직관주의 논리** ↔ 일반적인 타입이 있는 람다 계산법

---

## 토포스의 예시

### 1. Set — 집합의 카테고리

가장 친숙한 토포스. Ω = Bool, 고전 논리.

### 2. Presheaf 토포스 [C^op, Set]

어떤 카테고리 C에 대해 presheaf 카테고리는 항상 토포스다.

- **요네다 매장**으로 C를 이 토포스에 매장 가능
- Ω는 C의 체(sieve)의 presheaf

### 3. 유한 집합의 카테고리 FinSet

유한한 버전의 Set. 역시 토포스.

---

## 프로그래밍에서의 토포스

토포스적 사고의 프로그래밍적 의미:

1. **타입 시스템 = 직관주의 논리**: 프로그램은 증명, 타입은 명제
2. **부분 대상 분류자 = 술어 타입**: `a -> Bool`은 a의 부분 집합을 나타냄
3. **모든 유한 극한 + 지수**: 곱 타입, 함수 타입이 모두 존재 — 충분히 풍부한 타입 시스템

---

## 정리

- **토포스**: 유한 극한 + 지수 + 부분 대상 분류자 Ω를 갖춘 카테고리
- **부분 대상 분류자** Ω: 부분 집합(단사 사상)을 특성 사상으로 분류
- **Set에서**: Ω = Bool, true :: () → Bool
- **토포스의 논리 = 직관주의 논리**: 배중률이 반드시 성립하지 않음
- **Presheaf 토포스** [C^op, Set]: 항상 토포스. Ω = 체의 presheaf
- **Curry-Howard**: 토포스의 내부 논리 = 타입이 있는 람다 계산법
- 토포스는 "집합론의 일반화"로, 다양한 수학적 세계를 표현할 수 있다
