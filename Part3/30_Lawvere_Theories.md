# Chapter 30: Lawvere Theories — 로비어 이론

> **핵심 개념**: **로비어 이론(Lawvere Theory)**은 대수적 구조(모노이드, 군 등)를 정의하는 또 다른 방법이다. 모나드가 자기 함자 + 자연 변환으로 대수를 정의하는 반면, 로비어 이론은 **유한 곱을 가진 카테고리**로 대수를 정의한다. 이 두 접근법은 많은 경우 동등하지만, 각각 다른 통찰을 제공한다.

---

## 대수적 이론

"대수적 구조"를 정의하는 방법은 여러 가지다:

1. **집합론적**: 집합 + 연산 + 공리
2. **모나드적**: 자기 함자 T + η + μ, T-대수
3. **로비어 이론**: 유한 곱을 가진 카테고리 + Set으로의 곱 보존 펑터

로비어의 접근: 연산의 **"아리티(arity)"**(인자 수)를 카테고리의 대상으로, 연산 자체를 사상으로 표현한다.

---

## 로비어 이론의 정의

**로비어 이론** L은 다음을 갖춘 카테고리다:

1. **대상**: 자연수 0, 1, 2, 3, ... (각각 n은 "n개의 인자"를 나타냄)
2. **n은 1의 n-번 곱**: `n ≅ 1 × 1 × ... × 1` (n번)
3. **사상**: n → m은 "m개의 n-ary 연산"을 나타냄

특별한 펑터 I: **FinSet^op** → L이 존재하여 대상에 전사적(bijective on objects).

### L의 모델 (L-대수)

L의 **모델**은 곱을 보존하는 펑터 M: L → Set:

- `M(1)` = 기저 집합 (운반 집합)
- `M(n)` = M(1)^n (n은 1의 n-곱이므로)
- L의 사상 `f :: n → 1`은 n-ary 연산: `M(1)^n → M(1)`

---

## 예시: 모노이드의 로비어 이론

모노이드의 연산:
- **단위**: 0-ary 연산 `e :: 0 → 1`
- **곱**: 2-ary 연산 `μ :: 2 → 1`

공리 (사상의 동등성으로 표현):
- **결합법칙**: `μ ∘ (μ × id) = μ ∘ (id × μ) :: 3 → 1`
- **단위법칙**: `μ ∘ (e × id) = id = μ ∘ (id × e) :: 1 → 1`

모노이드의 로비어 이론의 모델 M은:
- `M(1)` = 집합 S
- `M(μ)` = S × S → S (이항 연산)
- `M(e)` = () → S (단위 원소 선택)
- 공리가 자동으로 만족

## 로비어 이론 vs 모나드

| 측면 | 로비어 이론 | 모나드 |
|------|------------|--------|
| 정의 | 유한 곱 카테고리 + 곱 보존 펑터 | 자기 함자 T + η + μ |
| 연산 | n → 1 사상 (n-ary 연산) | T의 구조에 암묵적 |
| 합성 | 카테고리의 합성 | 클라이슬리 합성 |
| 효과 | 순수 대수만 (부수 효과 제외) | 효과 포함 가능 |
| 일반성 | 유한 아리티만 | 무한 아리티 가능 |

**핵심 차이**: 로비어 이론은 **유한 아리티 연산**만 다루지만 더 "구문론적"이다. 모나드는 더 일반적이지만 연산의 구조가 덜 명시적이다.

### 동등성

많은 경우 로비어 이론과 (유한적으로 표현 가능한) 모나드는 **동등하다**:

- 로비어 이론의 모델 카테고리 ≅ 대응하는 모나드의 아일렌베르크-무어 카테고리
- 유한적(finitary) 모나드 ↔ 로비어 이론: 1:1 대응

## 정리

- **로비어 이론**: 대수적 구조를 유한 곱 카테고리로 표현
- **대상 n** = n개의 인자. **사상 n → 1** = n-ary 연산
- **모델** = 곱 보존 펑터 L → Set. 기저 집합 + 연산의 구체적 해석
- **모노이드의 로비어 이론**: 0-ary 연산 (단위) + 2-ary 연산 (곱) + 공리
- **로비어 이론 ↔ 유한적 모나드**: 많은 경우 동등
- 로비어 이론은 연산의 아리티를 명시적으로 다루어 더 "구문론적"
- **대수적 효과**: 로비어 이론의 현대적 프로그래밍 응용
