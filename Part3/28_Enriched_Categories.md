# Chapter 28: Enriched Categories — 풍성한 카테고리

> **핵심 개념**: 일반 카테고리에서 홈셋은 **집합(Set)**이다. **풍성한 카테고리(Enriched Category)**에서는 홈셋을 집합 대신 다른 **모노이드 카테고리(monoidal category)**의 대상으로 대체한다. 이를 통해 사상 사이의 "거리", "관계의 진리값", "사상들의 타입" 등 더 풍부한 구조를 표현할 수 있다.

---

## 왜 풍성한 카테고리인가?

일반 카테고리에서:
- **홈셋** C(a, b)는 집합
- **합성**은 함수: C(b, c) × C(a, b) → C(a, c)
- **항등**은 원소: id_a ∈ C(a, a)

하지만 때로는 사상들의 "집합" 이상의 구조가 필요하다:
- 사상 사이의 **거리**를 재고 싶다 → 홈 대상이 비음수 실수
- 사상의 존재 여부만 알고 싶다 → 홈 대상이 진리값 (True/False)
- 사상이 다른 카테고리의 대상이다 → 홈 대상이 그 카테고리의 대상

---

## 모노이드 카테고리 (Monoidal Category)

풍성한 카테고리를 정의하려면 먼저 **모노이드 카테고리** (V, ⊗, I)가 필요하다:

- **V**: 카테고리
- **⊗**: V × V → V (이항 펑터, "텐서 곱")
- **I**: V의 대상 (텐서 곱의 단위)
- 결합법칙, 단위법칙이 자연 동형으로 만족

예시:
- **(Set, ×, {*})**: 집합, 곱, 단원소 집합
- **(Cat, ×, 1)**: 카테고리, 곱, 단일 대상 카테고리
- **([0,∞], ≥, 0)**: 비음수 실수, max(또는 +), 0 — 거리 공간

---

## V-풍성한 카테고리의 정의

모노이드 카테고리 (V, ⊗, I)에 대한 **V-카테고리** C는:

1. **대상**: 모임 Ob(C)
2. **홈 대상**: 각 쌍 (a, b)에 대해 **V의 대상** C(a, b) ∈ V
3. **합성**: V의 사상 `∘ :: C(b, c) ⊗ C(a, b) → C(a, c)`
4. **항등**: V의 사상 `j_a :: I → C(a, a)`

결합법칙과 단위법칙을 만족해야 한다 (가환 다이어그램으로 표현).

---

## 예시들

### Set-풍성한 카테고리 = 일반 카테고리

(Set, ×, {*})로 풍성하게 하면 일반적인 카테고리를 복구한다.

### 2-풍성한 카테고리 = 원순서 (Preorder)

V = **2** (진리값의 카테고리: {False ≤ True}, ∧, True)로 풍성하게 하면:

- C(a, b) = True 또는 False
- 합성: True ∧ True = True (a ≤ b이고 b ≤ c이면 a ≤ c)
- 이것은 정확히 **원순서(preorder)**!

### [0,∞]-풍성한 카테고리 = 일반 거리 공간

- C(a, b) = a에서 b까지의 거리
- 합성: C(b,c) + C(a,b) ≥ C(a,c) (삼각 부등식)
- 항등: C(a,a) = 0

이것이 Lawvere의 통찰: **거리 공간은 풍성한 카테고리**다.

---

## 풍성한 펑터와 자연 변환

V-카테고리 사이의 **풍성한 펑터** F: C → D는:

- 대상의 매핑: a ↦ F a
- **홈 대상의 사상**: `F_{a,b} :: C(a, b) → D(F a, F b)` (V의 사상)

합성과 항등을 보존해야 한다.

**풍성한 자연 변환** α: F → G는 각 대상에서 성분을 가지며, 자연성 조건을 V의 사상으로 표현한다.

---

## 풍성한 요네다 보조정리

일반 요네다 보조정리를 V-카테고리로 확장:

```
[C^op, V](C(-, a), F) ≅ F a
```

여기서 좌변의 자연 변환 집합 대신 V의 대상(끝으로 표현)을 사용한다.

---

## 정리

- **풍성한 카테고리**: 홈셋을 모노이드 카테고리 V의 대상으로 대체
- **모노이드 카테고리** (V, ⊗, I): 텐서 곱과 단위를 갖춘 카테고리
- **Set-풍성 = 일반 카테고리**, **2-풍성 = 원순서**, **[0,∞]-풍성 = 거리 공간**
- V-합성: `C(b,c) ⊗ C(a,b) → C(a,c)` — V의 사상으로 표현
- 풍성한 펑터, 자연 변환, 요네다 보조정리 등 핵심 개념이 모두 일반화
- Lawvere의 통찰: 거리 공간, 순서 등 다양한 구조가 풍성한 카테고리의 예
